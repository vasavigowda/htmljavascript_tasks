Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255. For example, rgb(255, 0, 0) is displayed as red, because red is set to its highest value (255) and the others are set to 0.To display the color black, all color parameters must be set to 0, like this: rgb(0, 0, 0).
